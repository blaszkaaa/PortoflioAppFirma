# Lukas.biz.pl - Strona firmy budowlanej

## Opis projektu

Profesjonalna strona internetowa firmy budowlanej działającej od 2008 roku. Projekt prezentuje kompleksowe usługi budowlane w nowoczesny i przejrzysty sposób.

## Funkcjonalności

- **Responsywny design** - strona działa perfekcyjnie na wszystkich urządzeniach
- **Optymalizacja SEO** - lepsze pozycjonowanie w wyszukiwarkach
- **Galerie projektów** - prezentacja realizacji z filtrowaniem
- **Formularze kontaktowe** - łatwy kontakt z firmą
- **Integracja z systemami** - połączenie z domkilukas.pl

## Technologie

- React 18
- TypeScript
- Tailwind CSS
- Next.js
- Framer Motion

## Struktura projektu

```
src/
├── components/     # Komponenty React
├── pages/         # Strony aplikacji
├── styles/        # Style CSS
└── public/        # Zasoby statyczne
```

## Instalacja

```bash
npm install
npm run dev
```
